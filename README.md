Este repositorio contiene mis proyectos personales de análisis de datos, donde aplico SQL, Power BI, Excel y Python para resolver distintos casos prácticos.
## Proyectos
- **Análisis de ventas con Power BI:** Limpieza y visualización de datos de ventas mensuales.
- **Consulta SQL sobre exportaciones:** Extracción y análisis de datos de comercio exterior.
- **EDA en Python:** Exploración de una base de datos de Kaggle sobre precios de alimentos.
## Herramientas
- SQL  
- Power BI  
- Excel  
- Python (Pandas, Matplotlib)
